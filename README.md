# Batch-Resizer-App
A desktop app to resize multiple images to multiple sizes all in one go.

## Getting Started
Download
```
git clone https://github.com/Lasyin/Batch-Resizer-App.git
```

Install Prerequisites
```
npm install
bower install bootstrap
```

Run
```
npm start
```
